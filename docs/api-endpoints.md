# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Events

- `GET /api/events`
- `POST /api/events`
- `GET /api/events/:id`
- `PATCH /api/events/:id`
- `DELETE /api/events/:id`

### Hosts

- `GET /api/hosts`
- `POST /api/hosts`
- `GET /api/hosts/:id`
- `PATCH /api/hosts/:id`
- `DELETE /api/hosts/:id`

### Resorts

- `GET /api/resorts`
- `POST /api/resorts`
- `GET /api/resorts/:id`
- `PATCH /api/resorts/:id`
- `DELETE /api/resorts/:id`
